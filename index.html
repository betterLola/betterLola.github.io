安装注意事项

  • 目前仅支持 Windows 10/11 (64位)，Mac 版本开发中，敬请期待
  • 解压后请保持文件夹完整（包含 _internal 目录），不要单独移动 exe 文件
  • 首次运行如遇 Windows Defender / 安全软件拦截，请：
    ① 右键 exe → 属性 → 勾选"解除锁定"
    ② 或将整个文件夹添加到杀毒软件排除列表
  • 建议放置路径避免特殊字符（中文路径可以，但不要有空格或符号）
  • 多台电脑可用同一压缩包，无激活限制
  • 程序同目录会生成 SQLite 数据库记录吐槽历史，可放心删除重置

常见问题

被安全软件拦截怎么办？
这是正常现象，因为程序使用了全局热键监听（pynput库）。解决方案：
① 右键exe → 属性 → 勾选最下方"解除锁定" → 确定
② Windows安全中心 → 病毒和威胁防护 → 管理设置 → 排除项 → 添加该文件夹
③ 如果是360/腾讯管家：隔离区找到文件 → 恢复 → 添加信任
运行报错 "找不到DLL" 怎么办？
请确保目录结构完整：怨气宠物文件夹下必须同时存在 怨气宠物.exe 和 _internal 文件夹（同一级目录），不要删除或移动 _internal 目录。
会自动更新吗？数据保存在哪里？
不会自动更新，这是独立版本。如需最新版请关注TRAE论坛帖子或联系开发者(QQ: 387706580)。数据保存在程序同目录下的 grudge_pet.db SQLite数据库文件中。
有 Mac 版本吗？
目前仅提供 Windows 版本，Mac 版正在开发中。如急需 Mac 版可直接联系开发者反馈需求。
